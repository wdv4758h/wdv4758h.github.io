攝相座標系統（Viewing Coordinate System）

由照相機位置和取鏡方向所決定的座標系統。
設定 3D 物件模型空間位置時使用的座標系統

  • Windows Coordinates就是以螢幕的左上角的點為(0,0)，一直延伸到右下角

通常會透過這個座標來描述物體和光源的位置

  • Normalized Device Coordinates是將螢幕的正中央設定為(0,0,0)，做一個四邊體的3D空間

  • Eye Coordinates則是以使用者的眼睛作為(0,0,0)，在螢幕後方投影出一個四邊體的3D空間

  • Object Coordinates則是對每個不同的物件所使用的獨立座標系統
