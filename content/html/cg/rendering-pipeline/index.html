Per-Vertex

單位 : Polygon

viewing transformation

轉成 Viewing Coordinate System (攝相座標系統)(由照相機位置和取鏡方向所決定的座標系統)

projection transformation

轉成 Normalized Device Coordinates (將螢幕的正中央設定為(0,0,0)，做一個四邊體的3D空間)

Phong reflection model

[compute lighting]

單位 : 每個 vertex

phong reflection =

ambient (環境光) +

diffuse (物體在光源下的實際表現) +

specular (當入射光對物體產生全反射的情形)

是個 simple model，所以 只考慮 local lighting (只針對第一次反射做diffuse term 和 specular term 的計算)

diffuse term: Id = Ii * Kd * cosθ                (cosθ = L ‧ n)

specular term: Is = Ii * Ks * cosnsϕ = Ii * Ks * (R‧ V)ns

ambient term: I = Ia * Ka

Diffuse term represents object color

specular term represents light color

Back-face culling

perform in normalized coordinate ( or clip coordinate)

Perform

normal vector N =(A, B, C)

viewing direction vector V(0, 0, -1) (orthogonal projection)

the dot product of N and V is C

if C < 0, the polygon is back face

else the polygon is front face
