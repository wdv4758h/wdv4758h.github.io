View transformation

world coordinate 轉成 view coordinate

  • For fast perspective projection (simple formula) and clipping (simpler plane equation)
  • 包含 translation and rotation
  • Tview = B T

View projection

view coordinate 轉成 screen coordinate

view volume 轉成 normalized view volume (接著做 orthogonal projection)

TP = Tpers2 Tpers1

做 orthogonal projection

(Easy for hardware 3-D clipping)

  • Clipping can be unified for perspective and parallel projection
  • 包含 translation, shearing, and scaling

View-volume culling w.r.t. view volume in world space

View-volume culling w.r.t. normalized view volume in screen space

Window-to-Viewport Mapping

Window-to-viewport mapping

  • Map the window on the view plane to a viewport on the screen
  • 包含 translation, shearing, and scaling

3-D Geometric Transformations

  • translation (位移) : 非線性轉換，不可直接為矩陣  (要用 Homogeneous Coordinates)
  • scaling、rotation、shearing : 為線性轉換，可直接換為矩陣

Scaling (縮放)

性質 : linear, non rigid-body

Translation (平移)

性質 : affine, not linear, rigid-body

x’= x + 3

Rotation

性質 : linear, rigid-body

Shearing

性質 : linear, non rigid-body

在某一軸上進行 shear，其他軸保持不變

x’ = x + y cot(θ)

Affine transformation (仿射變換)

  • linear : x’ = L x
  • affine : x’ = L x + b

保留了點之間的 affine combinations

內部點轉換後之位置可由轉換後的端點算成 (要針對每個 vertex 處理)

保留線與線的平行關係、之間的比例，不維持 長度、角度

Rotation, scaling, shearing, translation

Homogeneous Coordinates (齊次座標)

因為需要一個 uniform matrix representation 來表示一連串的 affine transformations

series of affine transformations -> uniform matrix

Cartesian space -> homogeneous space

(x, y, z) -> (X, Y, Z, W), x = X/W, y=Y/W, z=Z/W, W != 0

Concatenation of transformations

Matrix 套用後會是反的

P -> A -> B -> C -> Q        =>        P -> CBA -> Q

Rotation about a fixed point

M = T(p) Rz(θ) T(-p)

Rotation about Origin

R = Rx Ry Rz

Rotation about arbitrary point and line

M = T(p0) Rx(-θx) Ry(-θy) Rz(-θz) Ry(θy) Rx(θx) T(-p0)
