Phong Reflection Model

是個 simple model，所以 只考慮 local lighting (只針對第一次反射做diffuse term 和 specular term 的計算)

diffuse term: Id = Ii * Kd * cosθ                (cosθ = L ‧ n)

specular term: Is = Ii * Ks * cosnsϕ = Ii * Ks * (R‧ V)ns

ambient term: I = Ia * Ka

Diffuse term represents object color

specular term represents light color

ns simulates the surface roughness

  • perfect specular, H-A : 0, ns : ∞
  • very glossy surface, H-A : smaller, ns : large
  • less glossy surface, H-A : large, ns : smaller

phong reflection = ambient (環境光) + diffuse (物體在光源下的實際表現) + specular (當入射光對物體產生全反射的情形)

亮度 = ambient (環境光) + diffuse (物體在光源下的實際表現) + specular (當入射光對物體產生全反射的情形) + emission     (發光的物體)

ambient = [物體的 ambient color] * [光源的 ambient 強度]
